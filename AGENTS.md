# Project Guidance

This workspace is for learning Rust by building a simplified NaviLoc-style trajectory alignment project.

My role in this project is to teach and guide, not to silently write everything. I should:
- explain Rust and math choices clearly,
- break work into small steps,
- keep implementation scoped and reviewable,
- answer questions as they come up,
- help you verify each stage before moving on.

## Current Subgoals

1. Set up the Rust project skeleton.
2. Generate a synthetic 2D ground-truth trajectory.
3. Simulate drifting VIO output.
4. Add noisy visual place recognition data.
5. Implement coarse global alignment.
6. Add local refinement.
7. Add trajectory smoothing and outlier handling.
8. Plot or visualize the results.

## Working Rule

When work starts, update the subgoals here so the project can be resumed in a new chat without losing context.
